Welcome to your site admin

FIRST: SET UP YOUR GITHUB TOKEN

You need this to publish changes to your live site. It only takes a minute and you only do it once.

  1. Go to github.com/settings/tokens (Fine-grained tokens)
  2. Click "Generate new token"
  3. Name it "UPVS Admin", set expiration to 1 year
  4. Repository access: "Only select repositories" and pick unprofessionalvarietyshow
  5. Permissions: set Contents to "Read and write"
  6. Click "Generate token" and copy it
  7. Come back here, hit Publish, paste the token

The token is stored in your browser. You only need to do this once.

Here's how the admin works:

EDITING CONTENT

Click any text on the page to edit it directly. Hover over images to replace them. Changes save automatically to a local draft in your browser.

SHOWS

Click "+ New Show" in the top bar to add a show. Click any show card to edit it. Add performers, recap photos, videos, and mark a show as upcoming.

SETTINGS

Click "Settings" in the top bar for fonts, colors, venue details, links, and press links. Changes preview live as you type.

PUBLISHING

When you're ready to go live, click "Publish". You'll paste your token the first time. After that, it's one click.
